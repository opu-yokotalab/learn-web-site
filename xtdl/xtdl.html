この文書について

概要

教材記述言語X-TDL(XHTML based Teaching material Description Language)の仕様書の草案です。

この文書の目的

X-TDLの記述を定めることです。この文書に書かれているものがX-TDLの正式な記述とします。X-TDLで教材を記述する場合や、X-TDLの処理系を作成する場合はこの文書に従ってください。なお、今後研究を進めるに従って、この文書の内容は変化する恐れがあります。

テストにおいて，出題する問題の問合せを記述するための要素です。

  • mark :グループへの配点

子要素

  • selection_type
  • selection_number
  • selection_point
  • ordering

selection_id要素

問題候補の指定のための要素。値として問題候補の識別子、または｢random｣を与える。

属性

  • 無し

子要素

  • 無し

要素値

  • 問題候補の識別子(複数指定の場合はカンマで区切る)
  • random

selection_type要素

問題候補の形式を指定するための要素です。

属性

  • 無し

子要素

  • 無し

要素値

  • 問題候補の形式ですか？

selection_number要素

出題する問題数を指定する。

属性

  • 無し

子要素

  • 無し

要素値

  • 出題問題数

selection_point要素

問題候補の持つ点数を指定する。指定した点数以上，以下の指定を行う。

属性

  • type　:｢over｣or｢under｣のいずれか

子要素

  • 無し

要素値

  • 点数

ordering要素

出題の順序を指定する要素。順番，またはランダムのいずれかを指定する。

属性

子要素

  • 無し
